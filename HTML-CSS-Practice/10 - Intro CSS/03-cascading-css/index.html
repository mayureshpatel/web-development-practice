Pokemon

Pokemon is is a Japanese media franchise managed by The Pokémon Company, a company founded by Nintendo, Game Freak, and Creatures. The franchise was created by Satoshi Tajiri in 1995, and is centered on fictional creatures called "Pokémon", which humans, known as Pokémon Trainers, catch and train to battle each other for sport.

The franchise began as Pocket Monsters Red and Green, a pair of video games for the original Game Boy handheld system that were developed by Game Freak and published by Nintendo in February 1996.
